## 📋 What This Website Does

This is **NOT** an online shopping/e-commerce website. It is a **B2B Export Business Catalog** website that:

- Showcases Indian export products (spices, ayurveda, handicrafts, agricultural goods, etc.)
- Allows international buyers to browse product categories
- Provides an enquiry/quote request system for bulk orders
- Displays company certifications (APEDA registered)
- Provides company contact information for trade inquiries
- Targets international importers looking for Indian goods
